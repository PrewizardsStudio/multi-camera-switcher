# Multi Camera Switcher
Easily switch between cameras in Maya
## Installation
Download the forceDeleteNodes.py file and just Drag & Drop into the Maya viewport. The scipt automatically download the icon and creates a new button on active shelf.
## Usage
In the window you see the list of scene's cameras. You can easily switch between them when select one. The tool store previous camera's current frame and next time when you select that camera recall this frame.
