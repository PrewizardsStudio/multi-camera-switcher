# Multi Camera Switcher
Easily switch between cameras in Maya
## Installation
Download the multiCameraSwitcher.py file and just Drag & Drop into the Maya viewport. The scipt automatically download the icon and creates a new button on active shelf.
## Usage
In the window you see the list of scene's cameras. You can easily switch between them when select one.<br>
The tool stores camera's current frame and range and next time when you switch to that camera it will recall them. You can turn on this recall functions in camera list's right click menu.
